# Final Report Project

This repository contains the files for our final report assignment.

## Project Overview

The project focuses on building energy modeling and heat transfer analysis. It includes simulation input files, model visualizations, supporting Python scripts, and the final report notebook.

## Repository Contents

- `Final_report_Ghosn_Stickl_Bengtstrom_Granfelt.ipynb`  
  Main Jupyter notebook for the final report.

- `requirements.txt`  
  Python package requirements needed to run the notebook and scripts.

- `dm4bem.py`  
  Supporting Python module used for building energy modeling calculations.

- `FRA_Marseille.07 6500_WEc.epw`  
  Weather data file used for the simulation.

- `Building.png`  
- `Heat_transfer_model.png`  
- `Model2.png`  
- `Model_of_building.png`  
- `Model_of_building_1.png`
  Images used for describing buildning and heat transfer models.
